Потік світла:

мистецтво освітлення

Цей курс надасть вам глибоке розуміння того, як працює світло. Як використовувати різні техніки освітлення для створення справжнього кіно

детальніше

Для кого цей курс?

Цей курс підійде для всіх, хто працює чи хоче працювати зі світлом, незалежно від рівня досвіду

Якщо ви хочете навчитися правильно виставляти світло для зйомок, створювати атмосферу в кадрі та робити професійний контент – цей курс для тебе

  • (01)

    Контент-креатори

    • Навчитесь керувати світлом у різних умовах: студія, вулиця, штучне та природне освітлення

    • Дізнаєтесь, як створювати кінематографічну картинку за допомогою світла

    • Освоїте популярні схеми освітлення для інтерв’ю, реклами, кліпів, художніх і комерційних проєктів

  • (02)

    Знімаєте для себе

    • Зрозумієте, як світло допомагає розповідати історію, створювати настрій і акцентувати увагу

    • Дізнаєтесь прості та доступні схеми освітлення для домашньої студії

    • Навчитесь робити якісний контент навіть без дорогого обладнання

  • (03)

    Початківці, які хочуть навчитися працювати зі світлом

    • Отримаєте чітке розуміння основ освітлення

    • Дізнаєтесь, яке обладнання варто купувати, а без чого можна обійтися

    • Навчитесь використовувати доступні джерела світла для створення професійного ефекту

Програма

  • ● Блок 1/4

    База & Основа

    Ти розберешся з усією технічною частиною, відчуєш свободу в творчості, навчишся розуміти та використовувати всі знання на практиці

    • Налаштування телефону для якісних відео
    • Камера в твоїх руках. Чітка та якісна картинка
    • Джерела світла: мечі, панелі, ліхтарі, студійне світло
    • Збираємо свій сет техніки для зйомок

  • ● Блок 2/4

    Світловий почерк

    Гра світла та тіні

    • Види світла: моделююче, заповнююче, мотивуюче, контрове, фонове, акцентне
    • Зйомка вдома без професійного обладнання
    • Атмосферні домашні відео з дешевим обладненням
    • Мистецтво студійного освітлення. Прості та складні рішення

  • ● Блок 3/4

    Знімай справжнє кіно

    Кіношне світло: як створити атмосферу в кадрі

    • Працюємо з димом
    • Драматична зйомка з тіньової сторони
    • Навчишся майстерно вибудовувати кадр
    • Цікаві світлові схеми
    • Повторюємо світло з фільмів
    • Кіношна обробка та пресети в безкоштовних додатках

● Блок 4/4

Випускна робота

Адаптуй всі знання під себе та зніми свою найкращу роботу
